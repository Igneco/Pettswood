FAQ

I hate your colour scheme. You are graphically inept. Make it better.

If you are a CSS guru, step right in and do what you think is best. This is HTML: it can look pretty, it can look official, it can be branded to match your organisation. JFDI.


It might be helpful to know that Pettswood unpacks the default pettswood.css when you first run a test. As soon as the file is there, it will not be touched again, so your changes are safe.
