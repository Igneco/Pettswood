FAQ

I hate your colour scheme. Can I do it differently?

You are obliged to. This is HTML: it can look pretty, it can look official, and it can be branded to match your organisation. You can and should structure your tests so that they are meaningful to the business, and easy to read.


Either modify pettswood.css, or add your own CSS file that overrides any classes you wish to change.


It might be helpful to know that Pettswood unpacks the default pettswood.css when you first run a test. As soon as the file is there, it will not be touched again, so changes you make are safe.
