Adding Pettswood to your sbt project

(1) Modify your build.sbt

// Add repo
resolvers += "Pettswood on Github" at "https://github.com/agmenc/Pettswood/tree/master/releases/"

// Add Pettswood library dependency
libraryDependencies += "org.pettswood" %% "pettswood" % "0.0.2" % "test" withSources()

// Tell sbt that the "test" action should run your Pettswood tests
testFrameworks += new TestFramework("org.pettswood.runners.sbt.PettswoodFramework")

(2) Create an empty class to hook into sbt's test integration

import org.pettswood.runners.SbtIntegrationHook

class PettswoodConfig extends SbtIntegrationHook
